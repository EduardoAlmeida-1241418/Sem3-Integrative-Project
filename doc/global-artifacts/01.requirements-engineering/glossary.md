# Glossary

**Terms, Expressions and Acronyms (TEA) are organized alphabetically.**

| **_TEA_** (EN)                        | **_TEA_** (PT)                                 | **_Description_** (EN)                                                                                                                                                                                     |
|:--------------------------------------|:-----------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **2D-Tree**                           | **Árvore 2D**                                  | A k-dimensional tree (k=2) for spatial indexing, used for efficient geographic range and nearest-neighbor queries.                                                                                         |
| **acceleration**                      | **aceleração**                                 | A key characteristic of a locomotive referring to its rate of increase in speed.                                                                                                                           |
| **action**                            | **ação**                                       | Description of an operation performed by a user and recorded in the system log.                                                                                                                            |
| **action_date**                       | **data da ação**                               | Timestamp indicating when a logged action occurred.                                                                                                                                                        |
| **Administrator**                     | **Administrador**                              | A system user role with elevated privileges for configuration and management.                                                                                                                              |
| **Aisle**                             | **Corredor**                                   | A passageway in a warehouse between rows of storage bays.                                                                                                                                                  |
| **aisleNumber**                       | **número do corredor**                         | Identifier or number of an aisle within a warehouse.                                                                                                                                                       |
| **AllocatedInfo**                     | **Informação de Alocação**                     | Information about the allocation of items to orders, including quantity allocated.                                                                                                                         |
| **allocationId**                      | **id de alocação**                             | Unique identifier for an allocation record.                                                                                                                                                                |
| **Assign**                            | **Atribuir**                                   | To designate a resource (e.g., train, route) for a specific purpose or operation.                                                                                                                          |
| **Assemble**                          | **Montar**                                     | To put together various components (e.g., locomotives and wagons) to form a train.                                                                                                                         |
| **Backbone Network**                  | **Rede de Espinha Dorsal**                     | A minimal subset of a network (e.g., rail tracks) that connects all nodes with minimal total length or cost.                                                                                               |
| **Bay**                               | **Zona de Armazenamento**                      | A specific storage location within a warehouse aisle.                                                                                                                                                      |
| **bayNumber**                         | **número da bay**                              | Identifier or number of a storage bay within an aisle.                                                                                                                                                     |
| **Bellman–Ford Algorithm**            | **Algoritmo de Bellman–Ford**                  | A graph algorithm that computes shortest paths from a single source, capable of handling edges with negative weights and detecting negative cycles.                                                        |
| **Best Fit Decreasing (BFD)**         | **Melhor Ajuste Decrescente**                  | A bin packing heuristic that sorts items by decreasing size and places each item into the bin where it fits with the smallest remaining capacity.                                                          |
| **Betweenness**                       | **Intermediação**                              | A graph centrality measure quantifying the number of times a node lies on the shortest path between other nodes.                                                                                           |
| **Board**                             | **Quadro**                                     | A software component that displays information (e.g., synoptic board) in a console.                                                                                                                        |
| **BogieType**                         | **Tipo de Bogie**                              | The type of bogie (wheel arrangement) used in locomotives or wagons.                                                                                                                                       |
| **Bottlenecks**                       | **Gargalos**                                   | Points in a network (e.g., rail corridor) where capacity constraints limit throughput or flow.                                                                                                             |
| **Box**                               | **Caixa**                                      | A unit containing a quantity of a specific SKU, stored in a warehouse bay.                                                                                                                                 |
| **Boxcar**                            | **Vagão Coberto**                              | A type of enclosed freight wagon used for transporting general goods.                                                                                                                                      |
| **Building**                          | **Edifício**                                   | A structure within facilities, such as Warehouse or Refrigerated Area.                                                                                                                                     |
| **BuildingType**                      | **Tipo de Edifício**                           | A classification for buildings within facilities, such as Warehouse or Refrigerated Area.                                                                                                                  |
| **Caesar Cipher**                     | **Cifra de César**                             | A simple substitution cipher where each letter in the plaintext is shifted a fixed number of places down the alphabet.                                                                                     |
| **Capacity**                          | **Capacidade**                                 | The maximum flow or volume of trains that a connection or system can handle per unit time.                                                                                                                 |
| **capacity**                          | **capacidade**                                 | Maximum number of boxes that a bay can store.                                                                                                                                                              |
| **capacityBoxes**                     | **capacidade em caixas**                       | The maximum number of boxes a storage bay can hold.                                                                                                                                                        |
| **Cargo**                             | **Carga**                                      | Goods or products being transported.                                                                                                                                                                       |
| **CargoType**                         | **Tipo de Carga**                              | Classification of cargo types (GENERAL_GOODS, CONTAINERS, LIQUIDS, etc.).                                                                                                                                  |
| **category**                          | **categoria**                                  | A classification for items or products.                                                                                                                                                                    |
| **Centrality Measures**               | **Medidas de Centralidade**                    | Quantitative metrics that identify the most important nodes within a graph (e.g., betweenness, closeness).                                                                                                 |
| **Chemical Tank Car**                 | **Vagão-Tanque Químico**                       | A specialized tank car optimized for transporting chemicals with high purity specifications.                                                                                                               |
| **Chokepoints**                       | **Pontos de Estrangulamento**                  | Critical nodes or links in a network where congestion or failure can disproportionately impact overall system performance.                                                                                 |
| **Circular Buffer**                   | **Buffer Circular**                            | A data structure that uses a single, fixed-size buffer as if it were connected end-to-end, allowing efficient FIFO operations. Also known as Ring Buffer.                                                  |
| **Circulation lines**                 | **Linhas de Circulação**                       | Railway tracks within a terminal where wagons move to reach parking areas or unloading docks.                                                                                                              |
| **Closeness Centrality**              | **Centralidade de Proximidade**                | A graph centrality measure based on the average shortest path distance from a node to all other nodes in the graph.                                                                                        |
| **Containers**                        | **Contentores**                                | Standardized reusable steel boxes used for intermodal freight transport.                                                                                                                                   |
| **Cost**                              | **Custo**                                      | A metric for a railway edge combining distance, capacity, congestion, and other operational penalties/bonuses.                                                                                             |
| **Customer Remorse**                  | **Arrependimento do Cliente**                  | A return reason where the customer returns an item that is still in good condition.                                                                                                                        |
| **Cycle**                             | **Ciclo**                                      | In graph theory, a path that starts and ends at the same vertex.                                                                                                                                           |
| **Cycle Count**                       | **Contagem de Ciclo**                          | A return reason related to inventory counting discrepancies.                                                                                                                                               |
| **Cycle Detection**                   | **Deteção de Ciclos**                          | The process of identifying cycles (paths that start and end at the same vertex) in a graph, important for topological sorting and negative cycle detection.                                                |
| **Damage**                            | **Danificado**                                 | A return reason where the product is physically broken or unusable.                                                                                                                                        |
| **Degree**                            | **Grau**                                       | In graph theory, the number of edges connected to a node (vertex).                                                                                                                                         |
| **Depot**                             | **Depósito**                                   | A facility for the storage, maintenance, or servicing of rolling stock.                                                                                                                                    |
| **Deterministic sweep**               | **Varredura Determinística**                   | A path sequencing strategy that sorts locations in a fixed order (e.g., ascending by aisle).                                                                                                               |
| **Dijkstra's Algorithm**              | **Algoritmo de Dijkstra**                      | A graph algorithm that finds the shortest paths from a source node to all other nodes in a graph with non-negative edge weights.                                                                           |
| **Dimensions**                        | **Dimensões**                                  | The measurable extents of an object, such as length, width, or height.                                                                                                                                     |
| **Directed Graph**                    | **Grafo Dirigido**                             | A graph where the edges have a direction, indicating a one-way relationship between nodes.                                                                                                                 |
| **Directed edge**                     | **Aresta Direcionada**                         | A graph edge with a defined direction, indicating a one-way relationship or dependency.                                                                                                                    |
| **Discard**                           | **Descartar**                                  | The action of disposing of returned goods deemed unsuitable for sale during quarantine inspection.                                                                                                         |
| **Dispatched State**                  | **Estado de Despachado**                       | The status of a train indicating it has been authorized to begin its journey.                                                                                                                              |
| **Dispatching**                       | **Despacho**                                   | The operational function of authorizing and controlling the movement of trains according to a schedule.                                                                                                    |
| **Disposition**                       | **Disposição**                                 | The outcome of a quarantine inspection, determining if goods are restocked or discarded.                                                                                                                   |
| **Dispatch**                          | **Despachar**                                  | To send of a train to its destination according to a schedule or plan.                                                                                                                                     |
| **Distance**                          | **Distância**                                  | The length of a connection between two nodes, typically measured in kilometers for railway edges.                                                                                                          |
| **DOT file**                          | **Ficheiro DOT**                               | A text file using the DOT graph description language for defining graphs for visualization tools like Graphviz.                                                                                            |
| **Double-track**                      | **Via Dupla**                                  | A railway segment with two parallel tracks, allowing simultaneous train movement in opposite directions.                                                                                                   |
| **dueDate**                           | **data de entrega**                            | The date by which an order must be fulfilled.                                                                                                                                                              |
| **Edge**                              | **Aresta**                                     | A connection between two nodes in a graph, representing a railway segment with associated attributes.                                                                                                      |
| **Edmonds–Karp Algorithm**            | **Algoritmo de Edmonds–Karp**                  | An implementation of the Ford–Fulkerson method for computing the maximum flow in a flow network, using BFS to find augmenting paths.                                                                       |
| **Electrified line**                  | **Linha Eletrificada**                         | A railway line equipped with overhead wires or a third rail to supply power to electric locomotives.                                                                                                       |
| **Eligible**                          | **Elegível**                                   | An order line status indicating its full requested quantity can be allocated from inventory.                                                                                                               |
| **EndpointEvent**                     | **Evento de Ponto Final**                      | An event recorded when a train arrives, departs, or passes at a facility.                                                                                                                                  |
| **entryDate**                         | **data de entrada**                            | The date when returned goods are placed in quarantine.                                                                                                                                                     |
| **EventType**                         | **Tipo de Evento**                             | The classification of an endpoint event (ARRIVAL, DEPARTURE, PASSAGE).                                                                                                                                     |
| **Expired**                           | **Expirado**                                   | A return reason where the product has passed its expiry date.                                                                                                                                              |
| **expiryDate**                        | **data de validade**                           | The date after which a product should not be used.                                                                                                                                                         |
| **Facility**                          | **Instalação**                                 | A place designed for a specific purpose, such as a freight terminal.                                                                                                                                       |
| **FacilityType**                      | **Tipo de Instalação**                         | A classification for railway facilities, such as Station, Terminal, or Freight Yard.                                                                                                                       |

| **Flow Network**                      | **Rede de Fluxo**                              | A directed graph where each edge has a capacity and receives a flow, used to model the maximum throughput between two points.                                                                              |
| **Freight**                           | **Carga**                                      | A set of wagons and their planned cargo that must be transported from an origin facility to a destination facility.                                                                                        |
| **Freight Manager**                   | **Gestor de Carga**                            | A user role responsible for managing freights and routes from a logistics perspective.                                                                                                                     |

| **frequency**                         | **frequência**                                 | The frequency of electrical power for electric locomotives.                                                                                                                                                |
